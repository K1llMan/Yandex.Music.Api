YandexMusicClient
==================================================================

Класс клиента для работы с API Яндекс.Музыки. Реализует основной функционал получения
объектов для взаимодействия с API. Для самих объектов функционал реализован в виде
методов-расширений.

------------------------------------------------------------------
Свойства
------------------------------------------------------------------

**Context**
   Account, с которым работает клиент.

   **Type**: YAccount

**IsAuthorized**
   Флаг авторизации

   **Type**: bool
